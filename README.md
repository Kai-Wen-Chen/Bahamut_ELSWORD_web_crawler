此爬蟲僅限場外艾爾之光第三串使用 可支援最新網站資料及本地資料搜尋

1. 選擇搜尋網站內容或是本地紀錄(如果本地沒有紀錄 會直接結束程式)
2. 若是搜尋網站內容 會詢問是否要限制搜尋的頁數範圍
3. 詢問關鍵字(若無則直接搜尋所有樓層)
4. 顯示出符合的樓層
5. 可將此次的搜尋存在本地(文件資料夾) 下次可直接使用本地紀錄節省時間
